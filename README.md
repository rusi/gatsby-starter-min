# Minimalistic Gatsby Starter Template
---

# Installation
---
Install this starter (assuming Gatsby is installed) by running from your CLI: 
```
gatsby new your-site-dir gh:rusi/gatsby-starter-min
```

# Running in development
```
gatsby develop
```
